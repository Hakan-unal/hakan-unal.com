Math, Date, String and Number Classes in JavaScript (With Samples)

JavaScript is a programming language specifically designed to make HTML pages more inreractive.
By using JavaScript you can make your website more interesting and more functional.
In addition, you can develop web, mobile and desktop application using many different libraries.
It is prederred by almost all programmers because it is easier to learn JavaScript and write code than other languages.
For this reason, JavaScript is one of the most used programming languages in the world. Creates the web infrastructure with HTML and CSS.
There are 4 classes on JavaScript. These are the:

  • Math
  • Date
  • String
  • Number
Let's begin to explain these classes with samples:

Math

The class used in mathematical operations in JavaScript is Math. Using the methods in the math class, you can easily perform operations such as square root and absolute value.
Below you can find samples of many methods of the Math class:

Math.random();
// Generates random numbers between 0 and 1

Math.sqrt(4);
// Operation: √4
// Output: 2

Math.pow(4,2);
// Operation: 42
// Output: 16

Math.abs(-5);
// Operation: |-5|
// Output: 5

Math.min(5,3,1,4,2);
// Operation: Find the smallest value
// Output: 1

Math.max(5,3,1,4,2);
// Operation: Find the biggest value
// Output: 5

Math.floor(3.7);
// Operation: Rounds the number to the smallest integer
// Output: 3

Math.ceiling(3.3);
// Operation: Rounds the number to the biggest integer
// Output: 4

Math.round(3.4);
// Operation: Rounds the number to the near integer
// Output: 1

Math.sin(x);
// Operation: Returns the sinüs value of the x angle

Math.cos(x);
// Operation: Returns the cosinüs value of the x angle

Math.E;
// Operation: Returns Euler's number

Math.PI();
// Operation: Returns PI value

Math.SQRT2
// Operation: Returns the square root of 2

Math.SQRT1_2
// Operation: Returns the square root of 1/2

Math.LN2
// Operation: Returns the naturel logarithm of 2

Math.LN10
// Operation: Returns the naturel logarithm of 10

Math.LOG2E
// Operation: Returns base 2 logarithm of E

Math.LOG10E
// Operation: Returns base 10 logarithm of E

Date

The Date class is used to handle date and time. It is quite difficult to do date operations directly in JavaScript. So, you may need to write very complex codes. For this reason, many libraries have been developed for date and time operations. Some of these libraries are as follows:

  • Moment.js
  • Date.js
  • XDate.js

Below you can find samples of many methods of the Date class:

getTime();
// Operation: Returns the time elapsed since January 1, 1970 (miliseconds)

getDate();
// Operation: Returns the day of the month (1-31)

getDay();
// Operation: Returns the day of the week.0 and 7 are considered Sunday

getMonth();
// Operation: Returns the month of the year (0-11)

getHours();
// Operation: Returns the hour of day (0-23)

getMinutes();
// Operation: Returns the minute of hour (0-59)

getSeconds();
// Operation: Returns the second of date (0-59)

setDate();
// Operation: Sets the day of the month (1-31)

setMonth();
// Operation: Sets the month of the year (0-11)

setFullYear();
// Operation: Sets the year of the date

setHours();
// Operation: Sets the hour of the date (0-23)

String

We don't need any classes to use the String class properties. We can apply it directly to any object whose data type is string.
Below you can find samples of many methods of the Date class:

let sentence = " AbCd aBcD ";

sentence.toUpperCase();
Output: " ABCD ABCD "

sentence.toLowerCase();
Output: " abcd abcd "

sentence.length;
Output: 11

sentence.trim();
Output: "AbCd aBcD"

sentence.bold();
Operation: Thickens the text

sentence.italic()
Operation: Makes the text italic

sentence.indexOf("aB")
Output: 6

sentence.search("Cd")
Output: 3

sentence.concat("" , "text");
Output: " AbCd aBcD text"

sentence.replace("AbCd" , "new text");
Output: " new text aBcD "

Number

Number properties and methods; provides some operations about numerical values
You can find common properties and methods of numerical values ​​below:

Number("123");
// Output: 123

parseFloat("1.123");
// Output: 1.234

toExponential(1.234);
// Operation: Converts to exponential number

const value = 1.2345;

value.toFixed(0);
// Output: 1

value.toFixed(2);
// Output: 1.23

value.toPrecision(2);
// Output: 1.2

value.toPrecision(3);
// Output: 1.23
